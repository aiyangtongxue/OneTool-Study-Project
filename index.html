📦 安装部署

环境要求：
  • PHP >= 8.0
  • ThinkPHP 伪静态规则
  • 运行目录设置为 public
1. 宝塔面板安装命令：
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh
2. Nginx伪静态配置：
location / {
    if (!-e $request_filename){
        rewrite  ^(.*)$  /index.php?s=$1  last;   break;
    }
}
